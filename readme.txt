■２．インストール方法
  １．「東方模倣風」はソニー非公式ゲームです。PSPで非公式ゲームを動作させる事は、
      アンソニーなので公式許可されていません。何らかの方法でセキュリティーを
      突破して下さい。

  ２．本アーカイブを解凍したファイルをメモリースティックの GAME 以下に配置します。
      ms0:/PSP/GAME/kene_r44/EBOOT.PBP (署名版実行ファイル)
      ms0:/PSP/GAME/kene_r44/data/... のような形です。(ms0:はメモリースティックの事です)
      (/kene_r44/の部分は余り文字数が長くなければ何でも良い。)

      kene_r44/非署名版/  (非署名版実行ファイル) 及び
      kene_r44/srcr44.zip (ソースコード) は、
      pspに入れる必要はありません。削除して構いません。
      ms0:/PSP/GAME/kene_r44/readme_r44.txt
      ms0:/PSP/GAME/kene_r44/231_readme_r44_test24.txt
      ms0:/PSP/GAME/kene_r44/231_readme_history_r44.txt
      は削除できます。
      (非署名版というのは、つまり-r34までの従来の方法です。)

  ３．OFW1.0(公式FW 1.00)の場合やCFWの場合等で、メモリースティックの容量が気になる場合は、
      kene_r44/非署名版/EBOOT.PBP (非署名版/実行ファイル) を
      ms0:/PSP/GAME/kene_r44/EBOOT.PBP に上書きして下さい。
      殆ど変わりませんが起動速度も若干速いです。

■３．予めご理解頂きたい事
  当ゲームに限らず、psp 用非公式ゲームは様々な非公式環境で動かされることに
  なります。その為、正常に動作しない、起動しない、等の問題が起こる
  事が多々あります。

  --------------------------------------------------------------
  動作環境
  --------------------------------------------------------------
  ◎対応 psp
     psp-1000(1, 2,  3,  4, 5, 6 )
     psp-2000(   2, (3), 4, 5, 6 )一部の後期版もCFW出来るらしい。
     psp-3000(           4, 5, 6 )CFW出来るらしい。
     psp-go  (           4, 5, 6 )CFW出来るらしい。(?)
     PS VITA (?)

  --------------------------------------------------------------
  ◎PSPで非公式ゲームを動作させるには、主に下記の方法があります。
    １．公式FW(ファームウェアー)1.00で実行する。(署名版はOFW1.00では動きません)
        (OFW == sony Original Firm Ware == 公式FW)
    ２．EBOOT.PBPを分割変換して、公式FW1.50で実行する。
    ３．kene_r44/非署名版/EBOOT.PBP (非署名版/実行ファイル) を使ってみる。
    ４．CFW(カスタムファームウェアー)やLCFW(ライトカスタムファームウェアー)を
        (LCFW == Light Custom FirmWare)
        導入(※)して、実行する。
        但し現時点で CFW や LCFW 導入不可能な psp本体 もある。
    ５．EBOOT LOADER を使って実行する。
    ６．HEN(Homebrew ENabler)を使って実行する。
    ７．HBL(Harf Byte Loader)を使って実行する。
    ８．他の署名アプリ(HBLとか)を介在して実行する。
    ９．他にもあると思うけど把握していません。

  CFW(カスタムファームウェアー)を導入(※)すると、(故障した場合等含め)
  ソニーのサポートが受けられなくなりますので、自己責任でお願いします。
  LCFWの場合は、良く知りません。m(__)m

  --------------------------------------------------------------
  このうち動作検証は、 1. と 3.(psp-1000/2000) と 4.(psp-1000/2000) しか
  していません。
  psp-3000/psp-goの場合は、有志の方がスレ(19)で(r35を動作)検証してくれて
  いるので、スレ(19)を良く読みましょう。
  --------------------------------------------------------------


  動作しなかった場合、まずは次のことを試してください。

    １．もう一度ダウンロードする。(稀です)
        稀にアーカイブが破損している場合があります。
        一度目にダウンロードしたアーカイブと
        二度目にダウンロードしたアーカイブ違いがあれば、
        どちらかが破損しているので破棄してください。

    ２．解凍ソフトを変えてみる。(稀です)
        r44では圧縮率は重視せず、なるべく多くの解凍ソフトが
        対応していそうなzip形式にしてみました。
        しかし全ての解凍ソフトが対応しているかはわかりません。

    ３．模倣風の配置を /GAMExxx/ に替えてみる。(xxxは３桁の番号)
        CFWの version によっては /GAME/ では、起動しなくて、
         /GAME500/ で起動したという例もあるそうです。(稀です)
        1. ms0:/PSP/GAME/kene_r44/data/... を
           ms0:/PSP/GAME500/kene_r44/data/... へ移動する。

    ４．EBOOT分割ツールを使い非署名版/EBOOT.PBPを 2 分割。(稀です)
           ms0:/PSP/GAME150/%kene_r44/EBOOT.PBP(サイズが小さい方)
           ms0:/PSP/GAME150/kene_r44/EBOOT.PBP(サイズが大きい方)
           ms0:/PSP/GAME150/kene_r44/data/... (その他データー全て)

  なるべく多くのpspで動作できるように、配慮して作成していますが、
  個人のため、サポートは期待しないで下さい。

■７．ＦＡＱ
    Ｑ．次の更新日はいつですか……
    Ａ．更新日について
      色々やってみましたが大変なので、あまり更新しない予定です。
      今後も更新しても大差ないと思われるので、更新日を気にするより
      御自身で改良して頂けると嬉しいです。
      更新は一年に一回ぐらいで十分な気もします。
      どうせ一年に一回更新ならば、せっかくなら一年で一番おめでたい日に
      しよう。そうしよう。という訳で、
    2015年は
      2015-01-01(木曜日) (r45)
    2016年は
      2016-01-01(金曜日) (r46)
    になります。(ただし予定は未定です)

    Ｑ．前の奴の方が面白かったんだけど……
    Ａ．そういう場合は、前の奴で遊んで下さい。
      ms0:/PSP/GAME/kene_r44/
      ms0:/PSP/GAME/kene_r43/
      ms0:/PSP/GAME/kene_r42/
        ...
      みたいになってても全然問題ありません。

    Ｑ．アナログキーがおかしいんだけど……
    Ａ．良く知りませんが「本体設定」→「本体情報」の画面で、
      アナログキーを上下左右に動かすと直るという話を聞いたことがあります。

    Ｑ．それでもアナログキーがおかしいんだけど……
    Ａ．模倣風起動時にアナログキーの入力があると、アナログキー破損と判断し
        模倣風はアナログキー無効モードになります。

    Ｑ．srcr44.zipってなにさ……
    Ａ．それはpspに入れる必要はありません。模倣風(EBOOT.PBP)を
        作成するための設計図(ソースコード)です。
        /非署名版/以下も同様に使わないものはpspに入れる必要はありません。

    Ｑ．32MBのメモカに入らないんだけど……
    Ａ．r38以降は32MBのメモリースティックDuoに対応していません。

    Ｑ．ブログとかで紹介したい……
    Ａ．インターネットは不特定多数の人が見ます。東方Projectを知らなかったり、
        誤解している人も見る訳です。ですから最低限の配慮は必要です。
        1. psp版の東方Project(公式)は現時点で存在しない。
           (あったらいいな)
        2. 模倣風は東方Projectではない。
           (ファンが作成した二次創作です。本家に迷惑をかけない様お願いします)
        3. 模倣風は現時点で開発中であり完成品ではない。
           (例え完成したとしてもお金を取る事は絶対に無い。オープンソースなので誰でも改造可能)
        のあたりが書いてあれば誤解されないんじゃないかな。

    Ｑ．東方模倣風ってどうしてこうなった？(AA略)
    Ａ．誰だって黒歴史の1つや2つありますでしょ？
